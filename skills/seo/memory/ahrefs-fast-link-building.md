## Core Concept
A step-by-step guide for launching a link building campaign quickly while building repeatable, scalable systems. The framework separates "finding prospects + contact info" from "outreach execution" to enable speed without sacrificing quality.

## Key Takeaways
- Speed comes from systematization, not shortcuts — build templates and processes first
- Find link prospects using competitor backlink analysis (Ahrefs Site Explorer → Backlinks)
- Use tools to find contact emails at scale (Hunter.io, Snov.io, VoilaNorbert)
- Automate outreach sequences but personalize the opening line of every email
- Track response rates per template to identify what's working
- Relationship-based links outperform cold outreach long-term
- Systematize so any team member can execute the same process

## Frameworks/Methods Covered
- **The 3-phase process**: Prospect research → Contact finding → Outreach sequencing
- **Systematization stack**: Ahrefs (prospects) + Hunter/Snov (emails) + Pitchbox/Mailshake (outreach)
- **Template optimization loop**: A/B test subject lines, track open rates, iterate weekly
- **Scaling approach**: Start with 20 outreach/day, scale to 100+ once templates are proven
