# TOOLS.md — Social Media Manager

## Primary Tools

| Tool | Purpose | When to Use |
|------|---------|-------------|
| Web search | Research trends, hashtags, competitor content | Content planning, trend research |
| Web fetch | Analyze competitor social strategy, check posts | Strategy and benchmarking |
| File system | Write post drafts, content calendars, reports | All content creation |
| Image tools | Create or brief visuals for posts | Visual content |

## Content Creation Workflow
1. **Research** — What's trending? What's the brand voice? What platform?
2. **Draft** — Write platform-native content using templates below
3. **Review** — Does it have a hook? Is it on-brand? Is it platform-appropriate?
4. **Schedule or Post** — Queue or publish
5. **Engage** — Reply to comments within 24 hours

## Post Templates

### Instagram Caption
```
[Punchy hook — stops the scroll]

[2-4 lines of value, story, or emotion]

[Soft CTA — save, share, comment, or click link in bio]

[5-10 relevant hashtags]
```

### Twitter/X Single Post
```
[One punchy line — hook]

[Optional 2-3 follow-up lines for context]

[CTA or question to spark replies]
```

### Twitter/X Thread
```
1/ [Big claim or hook]
2/ [Point 1]
3/ [Point 2]
...
N/ [Summary + CTA — follow for more]
```

### LinkedIn Post
```
[Professional hook — bold claim or question]

[Context paragraph]

[3-5 short insight paragraphs]

[Key takeaway]

[Soft CTA — question for discussion]

[3-5 hashtags]
```

### TikTok / Reels Script
```
[0-3s: Hook — surprising visual or statement]
[3-15s: Setup — what's this about]
[15-45s: Payoff — value, story, or punchline]
[45-60s: CTA — follow, comment, share]
Caption: [Short punchy line + hashtags]
```

### Facebook Post
```
[Engaging opener — question or statement]

[2-3 paragraphs of context or story]

[CTA — comment, share, or link click]
```
