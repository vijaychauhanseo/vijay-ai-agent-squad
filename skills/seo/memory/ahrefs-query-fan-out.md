## Core Concept
Query fan-out is a technique used by AI search platforms (Google AI Overviews, Perplexity, ChatGPT Search) that automatically expands a single user query into multiple sub-queries to generate more comprehensive answers. When someone searches "best AI tools for marketing", the AI may fan out into 5–10 hidden sub-queries before synthesizing an answer.

## Key Takeaways
- AI search doesn't just answer your query — it secretly runs dozens of related queries behind the scenes
- Pages optimized for one keyword may now need to address the full semantic cluster around that topic
- Query fan-out means a single piece of content can be cited for many different hidden sub-queries
- Traditional keyword targeting is no longer enough — topical authority and semantic depth win
- Pages need to answer not just the main question but the surrounding "why, how, what, who, when" questions
- AI platforms prefer sources that answer multiple related sub-queries on a single page
- Content that covers a topic comprehensively gets cited more often in AI-generated answers

## Frameworks/Methods Covered
- **Types of fan-out queries**: Clarification queries, decomposition queries, enrichment queries, verification queries
- **The fan-out → synthesis pipeline**: AI expands query → retrieves from multiple sources → synthesizes answer
- **How to optimize for query fan-out**: Build topical clusters, use FAQ sections, answer related questions within the same article
