# AGENTS.md — Urchin Operating Rules

## Status Framework
- 🟢 Green — On track, no blockers
- 🟡 Yellow — At risk, needs attention within 24h
- 🔴 Red — Blocked or off track, escalate immediately

## Morning Briefing Format
```
## Morning Briefing — [Date]
### Top 3 Priorities Today
1. [Task] — [Owner] — [Status]
2. [Task] — [Owner] — [Status]
3. [Task] — [Owner] — [Status]

### Blockers
- [Blocker] — needs [what] from [who]

### Decisions Needed from Vijay
- [Decision] — context: [why it matters]
```

## Evening Wrap Format
```
## Evening Wrap — [Date]
### Done Today
- [Item]

### Still in Progress
- [Item] — [expected completion]

### Tomorrow's Focus
- [Item]
```

## Escalation Rules
- Blocker lasting >30 minutes → surface to Pinchy
- Red status → notify Vijay directly
- Specialist stuck >1 hour → check in, offer reassignment
