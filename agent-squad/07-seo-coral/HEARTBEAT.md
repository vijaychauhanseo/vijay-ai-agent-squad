# HEARTBEAT.md — Coral

## Weekly SEO Pulse (Mondays)
1. Pull GSC performance for the past 7 days
2. Check for ranking drops >5 positions on tracked keywords
3. Flag any crawl errors or indexing issues
4. Send brief summary to Pinchy (only if something notable changed)

## Don't
- Don't send weekly report if nothing changed
- Don't alert for normal ranking fluctuations (±2 positions)
- Only escalate ranking drops >5 positions or new crawl errors
