## Core Concept
Ahrefs grew their YouTube channel from 10,000 to 200,000+ monthly views using keyword-driven content — not viral content or luck. The 3-step system prioritizes topics with proven search demand on YouTube, not just topics that "feel right."

## Key Takeaways
- YouTube is the 2nd largest search engine — treat it like Google SEO, not social media
- Start with a topic hierarchy, not individual keywords — prevents scattered content strategy
- YouTube search volume ≠ Google search volume — use Ahrefs' YouTube keyword data specifically
- Search intent on YouTube: tutorials, reviews, listicles, "how to" — match the format to the intent
- "Double dip" = some queries return both Google video carousels AND YouTube results — publish once, rank twice
- Low-competition YouTube keywords exist in abundance in tech/AI topics

## Frameworks/Methods Covered

**Step 1: Map a topic hierarchy**
- Pick 3-5 broad parent topics (e.g., "AI tools", "ChatGPT", "prompt engineering")
- For each parent topic, list 10-20 sub-topics — these become your video ideas
- Hierarchy prevents publishing random one-off videos

**Step 2: Generate keywords + find search volumes**
- Use Ahrefs Keywords Explorer → select YouTube as the search engine
- Check YouTube-specific search volume (different from Google)
- Filter by: Volume >100, KD <30 for quick wins
- YouTube autocomplete: type your topic + alphabet (a, b, c...) for more ideas

**Step 3: Check search intent**
- Watch the top 5 videos for your target keyword
- What format do they use? Tutorial, list, comparison, review?
- Match your video format to what's already winning

**Bonus: Double dip**
- Search your keyword on Google — does it show a video carousel?
- If yes: that keyword brings traffic from both Google and YouTube
- AI tutorial keywords almost always trigger video carousels
