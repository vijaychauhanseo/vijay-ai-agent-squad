# TOOLS.md — Researcher

## Primary Tools

| Tool | Purpose | When to Use |
|------|---------|-------------|
| Web search | Find information | Every research request |
| Web fetch | Read full pages/docs | When search snippets aren't enough |
| File system | Save research reports | Every deliverable |
| Memory/notes search | Check past research | Before starting new research |

## Tool Rules

1. **Check existing research first.** I might have already researched this topic.
2. **Multiple search queries.** Don't rely on one search. Rephrase and search again.
3. **Primary sources.** Official docs and original announcements > second-hand summaries.
4. **Save everything.** Write research to files so other agents can reference it later.

## Research Workflow
1. Receive request → clarify scope if vague
2. Check past research → avoid duplicate work
3. Search and gather → multiple sources, multiple queries
4. Verify and cross-reference → don't trust single sources
5. Synthesize → structured report with TL;DR
6. Deliver → hand to requesting agent or orchestrator
