## Core Concept
SEO visibility = the estimated % of total clicks available for your tracked keywords that you're actually receiving. A drop in visibility is an early warning signal before traffic actually falls. This guide covers how to detect, diagnose, and fix visibility losses.

## Key Takeaways
- SEO visibility ≠ traffic — visibility measures your share of the pie; traffic is the actual clicks
- Always compare visibility to organic traffic to confirm the drop is real
- Main causes of visibility loss: Google algorithm update, manual penalty, technical issue (crawl/index), content quality decline, increased competition
- Check Google Search Console first: any manual actions? Any coverage errors?
- Algorithm updates hit entire site categories — check if competitors in your niche also dropped
- Technical issues (accidental noindex, robots.txt block, canonicalization errors) cause sudden cliff drops
- Content quality issues cause gradual declines over months

## Frameworks/Methods Covered
**Diagnosis Flowchart**:
1. Confirm the drop in GSC (Impressions + Clicks)
2. Check date — does it align with a known Google algorithm update? (ahrefs.com/google-algorithm-updates)
3. Check for manual actions in GSC → Security & Manual Actions
4. Check technical: robots.txt, noindex tags, sitemap, crawl errors
5. Check content: did you change or delete pages recently?
6. Check competition: did a major competitor appear or improve?

**Fix by cause**:
- Algorithm update → audit content quality, E-E-A-T, thin pages
- Manual penalty → fix issue, file reconsideration request
- Technical → fix crawl/index issues immediately
- Competition → improve content depth, build more links
