| Tool | Purpose | When to Use |

| Image tools | Generate or describe visual assets | Asset creation |

## Design Workflow

1. **Brief** — Receive or write a creative brief (objective, audience, format, tone)
2. **Concept** — Define visual direction (mood, color, typography, layout approach)
3. **Spec** — Document exact specs (dimensions, colors HEX, fonts, spacing)
4. **Deliver** — Output asset descriptions, file specs, or actual files
5. **Handoff** — Pass to Charles (social), Developer (UI build), or Writer (content pairing)
6. **Review** — Check final output against brief and brand guidelines
