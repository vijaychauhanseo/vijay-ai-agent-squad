# TOOLS.md — Marketing / Growth

## Primary Tools

| Tool | Purpose | When to Use |
|------|---------|-------------|
| Web search | Research trends, competitors, audiences | Campaign planning |
| Web fetch | Analyze competitor marketing, check our pages | Strategy |
| File system | Write social posts, email drafts, reports | All content |
| Messaging | Post to channels (if integrated) | Distribution |

## Tool Rules

1. **Promote before creating new.** Check if existing content needs distribution before creating more.
2. **Repurpose first.** One piece of content = multiple formats. Extract all value.
3. **Platform-native.** Don't cross-post identical content. Adapt for each platform.
4. **Track everything.** If you can't measure it, you can't improve it.

## Social Post Templates

### Twitter/X
```
[Hook — one punchy line]

[2-3 lines of insight or story]

[CTA or question for engagement]

[link if relevant]
```

### LinkedIn
```
[Professional hook]

[3-5 short paragraphs with insights]

[Takeaway or lesson]

[Soft CTA — question or invitation to discuss]
```

### Email Newsletter
```
Subject: [Curiosity-driven, under 50 chars]

[Personal greeting]
[One key insight or update]
[Why it matters to the reader]
[CTA — one clear action]
```
