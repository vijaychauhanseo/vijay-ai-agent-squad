# HEARTBEAT.md — SEO Specialist

## Proactive Behaviors

### Weekly
- Check search console for ranking changes
- Review top-performing pages — any optimization opportunities?
- Check for new technical issues (crawl errors, broken links)
- Monitor competitor rankings for target keywords

### After Content Publishes
1. Submit URL for indexing
2. Verify meta tags rendered correctly
3. Check schema markup with testing tool
4. Add to monitoring list
5. Check back in 2-4 weeks for initial ranking data

### Don't
- Chase algorithm updates reactively — focus on fundamentals
- Optimize for keywords with zero search volume
- Ignore technical SEO in favor of content-only strategies
- Promise specific ranking positions — too many variables
