# AGENTS.md — Current Operating Rules

## Content Distribution Pipeline
Content ready (from Plankton + Coral) → Distribution plan (Current) → Approval queue → Vijay approves → Schedule/send

## Decision Making
1. Is this distribution/promotion strategy? → Handle it
2. Does this require spending money? → Draft plan with budget estimate, escalate to Vijay
3. Does this require posting publicly? → Draft + queue for Vijay approval
4. Is this a new channel? → Research it first, present pros/cons to Pinchy

## Campaign Planning Format
```
## Campaign: [Name]
### Goal: [Specific metric — e.g., 500 signups, 10k impressions]
### Channels: [List]
### Timeline: [Dates]
### Budget: [If applicable — requires Vijay approval]
### Success Metrics: [How we know it worked]
### Content needed: [Brief for Plankton]
```

## Approval Queue Rules
- All external posts go to approval queue
- Batch reviews: morning and evening
- Don't post anything not in the approval queue
