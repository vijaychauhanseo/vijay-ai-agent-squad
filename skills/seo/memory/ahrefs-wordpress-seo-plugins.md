## Core Concept
Ahrefs' tested list of 15 WordPress plugins that cover all major SEO needs: on-page optimization, performance, images, redirects, schema, and security. This is the WordPress SEO toolkit — install what's needed, not everything at once.

## Key Takeaways
- You don't need all 15 — pick one from each category (on-page, performance, images, redirects)
- Yoast SEO vs Rank Math: both are excellent — Rank Math is free with more features, Yoast has longer trust history
- Speed plugins (WP Super Cache + Autoptimize) directly impact Core Web Vitals and rankings
- Image optimization (ShortPixel) is often the single biggest performance win for image-heavy blogs
- Broken link checker is essential maintenance — dead links hurt UX and crawl efficiency
- HTTPS (Really Simple SSL) is a ranking signal — must be active

## The 15 Plugins by Category

**On-page SEO (pick one):**
1. **Yoast SEO** — industry standard, title/meta/XML sitemap/schema, traffic light UX
2. **All in One SEO Pack** — full-featured alternative to Yoast
3. **Rank Math** — free, advanced features (schema types, 404 monitor, keyword rank tracking)
4. **The SEO Framework** — lightweight, no bloat, for developers

**Content UX:**
5. **Easy Table of Contents** — auto-generates TOC from headings, improves UX + jump links

**Maintenance:**
6. **Broken Link Checker** — scans for 404s and broken outbound links automatically

**Performance:**
7. **ShortPixel** — bulk image compression (lossy/lossless), WebP conversion
8. **a3 Lazy Load** — defers off-screen image loading, boosts LCP score
9. **WP Super Cache** — page caching, dramatically reduces server response time
10. **Autoptimize** — minifies CSS/JS/HTML, reduces page size

**International:**
11. **HREFLANG Tags Lite** — manages hreflang for multilingual sites

**Technical Maintenance:**
12. **Redirection** — manage 301 redirects, track 404s
13. **Really Simple SSL** — activates HTTPS, fixes mixed content warnings

**Security/Quality:**
14. **Anti-spam** — removes comment spam that dilutes site quality signals
15. **Rel NoFollow Checkbox** — adds nofollow to outbound links easily
