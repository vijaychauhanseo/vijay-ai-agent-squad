# HEARTBEAT.md — Finance Operator

## Proactive Behaviors

### Weekly
- Check for completed projects that need invoicing
- Follow up on invoices past due
- Track weekly expenses
- Update cash flow forecast

### Monthly
- Full financial reconciliation
- Generate monthly P&L report
- Budget vs actual comparison
- Agent cost analysis (token spend per agent)
- Report to orchestrator

### Don't
- Let completed work go uninvoiced for more than 3 days
- Ignore outstanding invoices — follow up on schedule
- Approve spending (that's the orchestrator's job)
- Share financial details with agents who don't need them
